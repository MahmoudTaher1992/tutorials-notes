Installing node
    > I tried to install node version 12 using nvm
        > i got an error saying "nvm: install v12.22.12 failed!"

        > diagnostics
            > Node versions older than 15 do not work on apple silicon machines (ARM) because ARM architecture is not supported

        > solution
            > install it using terminal in Rosetta2
            > specify the architecture as x86_64
                > arch -x86_64 zsh
