Functional programming
    it is one of the programming paradigms (Functional, OO, Procedural, Event-driven)

    It is about decomposing the problem into small functions that take an input and return and output > compose those functions
    to solve the problem

    Pros

        Easier to debug
        Easier to test
        Scalable
