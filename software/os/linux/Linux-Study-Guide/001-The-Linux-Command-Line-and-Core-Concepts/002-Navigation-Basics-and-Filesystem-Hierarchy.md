### 1. Directory Hierarchy Overview (The Structure)
Linux follows the **Filesystem Hierarchy Standard (FHS)**. This ensures that a Linux user knows roughly where to find files, regardless of whether they are using Ubuntu, Fedora, or Debian. It is an inverted tree structure.

*   **`/` (The Root):**
    *   This is the top of the tree. Every single file and device on the Linux system lives under this "Slash."
    *   *Example:* You cannot go "higher" than root.
