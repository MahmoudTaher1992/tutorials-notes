Visibility
    > specifies whether or not an element is visible
    > values
        > visible
            > Default value. The element is visible
        > hidden
            > The element is hidden (but still takes up space)
        > collapse
            > Only for table rows (<tr>), row groups (<tbody>), columns (<col>), column groups (<colgroup>). 
            This value removes a row or column, but it does not affect the table layout. The space taken up 
            by the row or column will be available for other content
