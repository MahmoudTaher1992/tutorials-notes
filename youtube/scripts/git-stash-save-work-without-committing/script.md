# git stash: Save Work Without Committing

**Tone:** Serious / Educational
**Duration:** ~30 seconds

| Sentence | Type |
|----------|------|
| You are mid-feature and your teammate needs a hotfix right now. | Hook |
| Here is the one git command that saves your work without committing. | Title Reveal |
| Making a fake WIP commit just to switch branches pollutes your history. | Introduction |
| `git stash` snapshots your changes and leaves a clean working tree. | Body – Point 1 |
| Switch branches, fix the bug, then switch back whenever you are ready. | Body – Point 2 |
| `git stash pop` restores all your changes exactly as you left them. | Body – Point 3 |
| No fake commits, no lost work, no context juggling. | Conclusion |
| Next time you get interrupted mid-feature, reach for `git stash` first. | CTA |
